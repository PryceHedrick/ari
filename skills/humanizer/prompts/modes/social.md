# Mode: Social (NOVA — X/Twitter Content)

Social media content for X (Twitter), short-form posts.

## Voice

- Punchy, opinionated, shareable
- Hook in first line (no throat-clearing)
- Confidence without arrogance
- Educational + entertaining mix

## Structure

- Line 1: Hook (statement, question, or surprising fact)
- Lines 2-4: Supporting point or context
- Last line: Insight or call to reflect

## Platform Rules (X/Twitter)

- Under 280 chars for single tweet
- Thread: each tweet must stand alone
- No hashtag spam (0-2 max)
- No link in main tweet if possible (save for reply)

## Avoid

- "Excited to announce..."
- "Thrilled to share..."
- "Honored to..."
- Hollow inspirational filler

## Examples

Before: "We're excited to announce that we've achieved significant growth this quarter."
After: "Q1: 3x revenue. 0 VC money. Here's what actually worked:"
